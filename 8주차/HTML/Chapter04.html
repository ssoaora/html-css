4-01 텍스트 입력하기

hn 태그는 1~6의 크기가 있습니다.
h1 태그가 가장 큽니다.
일반 글자 크기는 h4 입니다.
자동으로 줄바꿈을 생성합니다.

p태그는 align 속성으로 텍스트 정렬을 할 수 있습니다.
현재 중앙 정렬을 하고 있습니다.

현재 양쪽 정렬을 하고 있습니다.

내부의 텍스트를 들여쓰기하여 다른 문장들과 구분될 수 있게합니다.
들여쓰기는 blockquote 태그를 사용합니다.



텍스트 굵게 표시하기

의미적으로 중요한 부분을 굵게 표시합니다.

단순하게 글자를 진하게 표시합니다.


텍스트 기울이기

의미적으로 중요한 부분을 강조하기 위해 이탤릭체로 표현합니다.

단순하게 글자를 기울여 표시합니다.


문자 바탕 강조와 밑줄

문자열의 바탕색을 노란색으로 지정

문자에 밑줄을 그어줍니다.

hr 태그

물리적 문단과 논리적 문단

물리적 문단

문장의 위와 아래에 빈 라인이 생성됩니다.
p 태그로 사용해 생성합니다.

논리적 문단

하나의 물리적 문단 내에 여러개의 논리적인 문단이 있을 수 있음.
문단의 위와 아래에 빈 라인이 생성되지 않음.
한 문장 내에서 일부분을 문단으로 지정 가능.
div 태그와 span 태그로 생성합니다.

div 태그

중첩해 사용가능. 즉, div 태그 내부에 또다른 div 태그 사용 가능.
block 요소임. 즉, 하나의 문장에 하나의 div만 존재. (자동으로 줄 바꿈이 발생함.)
너비와 높이 지정 가능.



font 태그

font 태그는 글자의 크기, 색상, 글꼴을 지정할 수 있습니다.
모든 속성을 지정해야 하는 것은 아닙니다.
